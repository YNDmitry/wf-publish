===============================
Webflow Publish Plugin Setup
===============================

Follow these instructions to set up and run the Webflow Publish Plugin.

1. Prepare the installation script:
   Make the `install.sh` file executable by running the following command:

   chmod +x install.sh

2. Run the installation script:
   Install the plugin by passing the `wf-publish.zip` archive as an argument:

   sudo ./install.sh wf-publish.zip

3. Create a new project:
   Use the following command to create a project:

   npx create-webflow-publish@latest

4. Log in to Webflow:
   Ensure you are logged in to your Webflow account in the browser you specified when creating the project.

5. Start the project:
   Run the following command to start the project:

   npm run dev

   If you are using a different package manager, replace `npm` with the appropriate one.

6. Done!
   The Webflow Publish Plugin is now set up and ready to use.
   Enjoy!
