===============================
Webflow Publish Plugin Setup
===============================

Следуйте этой инструкции, чтобы настроить и запустить Webflow Publish Plugin.

1. Подготовка скрипта установки:
   Сделайте файл `install.sh` исполняемым, выполнив следующую команду:

   chmod +x install.sh

2. Запуск скрипта установки:
   Установите плагин, передав архив `wf-publish.zip` в качестве аргумента:

   sudo ./install.sh wf-publish.zip

3. Создание нового проекта:
   Используйте следующую команду для создания проекта:

   npx create-webflow-publish@latest

4. Авторизация в Webflow:
   Убедитесь, что вы вошли в аккаунт Webflow в браузере, который вы указали при создании проекта.

5. Запуск проекта:
   Для запуска проекта выполните:

   npm run dev

   Если вы используете другой пакетный менеджер, замените `npm` на соответствующий.

6. Готово!
   Теперь Webflow Publish Plugin настроен и готов к использованию.
   Удачной работы!
===============================
